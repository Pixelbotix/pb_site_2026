Boot Camp Breakdown

A step-by-step hands-on journey designed for future innovators.

Day 1: Fun with Electronics & Circuits

  • • Learn basic electronic components like Resistors, LEDs, Sensors, and Buzzers.
  • • Master safe, solderless breadboard usage for building real hardware circuits.
  • • Project: Build Solar Powered LEDs and functional Clap Switch circuits.

Day 2: Coding Made Easy (Arduino)

  • • Understand Microcontroller fundamentals and the basics of robotic sensors.
  • • Familiarize with Block and Script based coding to control external hardware.
  • • Project: Implement SOS light patterns and smart Traffic Light systems.

Day 3: Build Your Own Mobile Robot

  • • Identify components and assemble a professional robot chassis from scratch.
  • • Mount high-precision Servo motors and Ultrasonic "Robot Eye" sensors.
  • • Outcome: Build a robot that navigates mazes and avoids obstacles autonomously.

Day 4: 3D Printing & Design Thinking

  • • Master 3D design and modeling using professional tools like TinkerCAD.
  • • Learn rapid prototyping concepts to solve complex real-world challenges.
  • • Project: Design and print custom keychains and favorite characters.

Day 5: Introduction to AI Tools

  • • Train Image Classifiers using Google Teachable Machine for automation.
  • • Explore generative AI for Art creation and building custom Chatbots.
  • • Learn to leverage AI safely to generate simple websites and applications.
